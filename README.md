# Garfie Chiu — Personal Portfolio

Source code for [garfiec.github.io](https://garfiec.github.io).

## Tech Stack

- **Vue 3** with `<script setup>` SFCs
- **TypeScript**
- **Vite**
- **SCSS**
- FontAwesome icons, tsParticles

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview  # preview the production build locally
```

## Deployment

Pushes to `master` automatically build and deploy to GitHub Pages via [GitHub Actions](.github/workflows/deploy.yml).
